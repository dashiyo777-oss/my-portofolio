## 貼り方（毎回同じ）

1. noteで新規記事 → 記事タイプを **「有料」** に設定
2. ファイル内の **【① 無料パート】** を本文の先頭に貼る
3. 無料パートの直後で **「有料エリア設定」の線**を引く（ファイル内の★の位置）
4. その下に **【② 有料パート】** を貼る
5. **メンバー特典マガジン「世界のよいニュース」** に追加
6. 公開 → シークレットウィンドウで「無料パートが見える → その先が会員限定」を確認
