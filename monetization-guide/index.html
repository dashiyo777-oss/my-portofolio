02

登場する道具（キャラクター）を
ひとりずつ紹介

それぞれの道具が「何屋さんなのか」を、身近なものにたとえて紹介します。名前は今おぼえなくて大丈夫です。

🤖Claude / 生成AI
＝ 何でも作れる「魔法の職人さん」

「こういうサイトを作って」と日本語でお願いすると、プログラム（お店の中身）を書いてくれます。ChatGPTやGeminiなど、他のAIでも同じことができます。

お金：無料でも使える。もっと使うなら月 約3,000円〜（Proプラン）
📦GitHub（ギットハブ）
＝ 作品をしまう「頑丈な宝箱」

作ったプログラムをネット上に保存する場所。「いつ・どこを変えたか」の記録も残るので、まちがえても前の状態に戻せます。Cloudflareと自動でつながります。

お金：基本ずっと無料
☁️Cloudflare（クラウドフレア）
＝ お店を出す「世界一等地の土地」

Pagesで作ったサイトを世界に公開し、Workersで「会員だけ見られる」などの賢い動きを追加できます。速くて、世界中どこからでも見られます。

お金：個人なら無料枠で十分。使い込むと月 約750円〜
🏷️ドメイン
＝ お店の「住所・名前」

example.com のような、あなただけの住所。無料の住所（〇〇.pages.dev）でも始められますが、自分の名前の住所があると本格的でカッコいい。

お金：1年で 約1,500円〜（なくても始められる）
💳Stripe（ストライプ）
＝ お店の「レジ・お会計係」

クレジットカードでの支払いを、安全に受け取る仕組み。「月額会員」「1回だけ購入」など、いろいろなお金の受け取り方ができます。

お金：登録は無料。売れたときだけ約3.6%の手数料
📣Google AdSense
＝ お店にはる「広告ポスター」

サイトに広告を貼ると、見られたりクリックされたりするたびに、少しずつお金がもらえる仕組み。お客さんが多いほど、もらえるお金が増えます。

お金：登録・利用は無料（審査あり）
03

作って公開するまでの
5ステップ

実際の流れはこの5つだけ。プログラムの知識がなくても、AIにお願いしながら1つずつ進められます。

1

アイデアを決める 💡

「どんな困りごとを解決するサイト・アプリか」を1行で決めます。ここがいちばん大事。小さくてOK。

例：「毎日ひとつ、健康の豆知識が読めるアプリ」「昔の偉人の名言でクイズができるゲーム」
2

AIに作ってもらう 🤖

ClaudeなどのAIに「こういうサイトを作って」とお願いします。できあがったコードをコピーして、パソコンに保存します。うまくいかない所は「ここをこう直して」と会話しながら直します。

この記事の「06 プロンプト集」に、そのままコピペで使えるお願い文を用意しています。
3

GitHubにしまう 📦

作ったファイルをGitHubの箱（リポジトリ）に入れます。これで作品が安全に保管され、次の公開ステップと自動でつながります。

4

Cloudflareで世界に公開 ☁️

CloudflareにGitHubの箱をつなぐと、あなたのサイトがインターネット上に出て、世界中の人が見られるようになります。次からはGitHubを更新するだけで、サイトも自動で新しくなります。

ここまでで、かかったお金は0円でも可能。もう「自分のサイトを持っている人」です！
5

お金を生む仕掛けをつける 💰

公開できたら、広告を貼ったり、有料の会員機能をつけたりして、収益化します。仕掛けの種類は次の章でくわしく説明します。

04

お金を生む「6つの仕掛け」

サイトからお金が生まれる方法はいくつもあります。代表的な6つを、むずかしさ（★の数）と一緒に紹介します。1つだけでなく、組み合わせてもOKです。

📣

① 広告を貼る

サイトに広告を表示して、見られたりクリックされるたびに少しずつお金がもらえます。いちばん始めやすい方法。

使う道具：Google AdSense
むずかしさ ★☆☆元手 0円
🔁

② 月額会員（サブスク）

「毎月◯円で、特別なコンテンツが見放題」という会員制。安定して続くお金になりやすいのが強み。

使う道具：Stripe ＋ Cloudflare Workers
むずかしさ ★★★手数料 3.6%
🛒

③ デジタル商品を売る

電子書籍・テンプレート・音楽・イラストなど、データの商品を1回売り切りで販売します。在庫がいらないのが利点。

使う道具：Stripe（決済リンク）
むずかしさ ★★☆手数料 3.6%
🔗

④ アフィリエイト

「この商品いいよ」と紹介し、あなたのリンク経由で買われたら紹介料がもらえる仕組み。記事やレビューと相性◎。

使う道具：Amazonアソシエイト 等
むずかしさ ★☆☆元手 0円
🎁

⑤ 投げ銭・応援

作品を気に入った人が「応援」としてお金を送れるボタンを置きます。ファンとの距離が近い作り手向け。

使う道具：Stripe / 各種サービス
むずかしさ ★☆☆元手 0円
💼

⑥ 実績にして仕事を得る

作ったサイト自体を「ポートフォリオ（実績集）」にして、「あなたにも作って」という制作の依頼につなげます。単価がいちばん大きくなりやすい。

使う道具：このポートフォリオそのもの
むずかしさ ★★☆単価 大
おすすめの順番

最初は ①広告 と ④アフィリエイト（どちらも元手0円）から。人が集まってきたら ②サブスク や ③商品販売 に広げ、実力がついたら ⑥仕事の受注 へ、と階段をのぼるのが王道です。

05

結局いくらかかるの？

結論から言うと、0円で始めて、うまくいってから少しずつお金をかけるのが正解です。下の表は、月ごとの目安（1ドル＝約150円で計算）です。

道具 役割 無料で始める 本格的にやる（月あたり）
Claude / 生成AI 作る職人 0円（無料版） 約3,000円（Pro）
GitHub 保存する箱 0円 0円（個人はずっと無料）
Cloudflare Pages / Workers 公開する土地 0円 約750円（Workers有料）
ドメイン 住所・名前 0円（無料の住所を使う） 約130円（年1,500円ほどを月割）
Stripe レジ・決済 0円 売れた分の3.6%だけ
Google AdSense 広告 0円 0円
合計 — 0円 でスタート可能 月 約 4,000円 ＋ 売上の手数料
たとえ話

お小づかい0円でも「お店ごっこ」を始められて、お客さんが増えてから、ちょっといい看板（ドメイン）やレジ（決済）を足していく——というイメージです。最初から全部そろえる必要はまったくありません。

① サイトをゼロから作ってもらう

08

よくある質問

プログラミングが全くできなくても大丈夫？

大丈夫です。今はAIがコードを書いてくれるので、あなたの仕事は「何を作りたいかを言葉で伝えること」と「できたものを試して感想を返すこと」。この2つができれば始められます。

本当に0円で始められるの？

はい。AIの無料版・GitHub・Cloudflare・AdSenseはすべて無料枠があります。ドメインや決済は「お金を稼げるようになってから」足せばよいので、スタート時点の出費は0円で問題ありません。

どのくらいで稼げるようになる？

人によりますが、収益化は「作ること」より「見にきてくれる人を増やすこと」が9割です。最初の数か月は種まきの期間。1つのサイトに固執せず、いくつか作って反応の良いものを育てるのが近道です。

ClaudeとChatGPT、どっちを使えばいい？

この記事の作り方は、どのAIでも通用します。長めのコードを一気に書いてもらう用途ではClaudeが得意ですが、まずは今使えるものでOK。慣れてきたら両方使って、良い方を選びましょう。

子ども（未成年）でもお金を受け取れる？

広告や決済サービスの多くは、契約に年齢の条件があります。未成年の場合は保護者の名義・同意が必要になることが多いので、収益化の前におうちの人と相談してください。作って公開するところまでは、だれでも楽しめます。
