# 効く（Kiku）

> 本当に効く生活の知恵を、ランクで。
> 健康・生活のお役立ち情報を「体感・科学的根拠・専門家監修」で評価し、ジャンル別にランク付けするWebアプリ（MVP）。

## 動かし方

追加のインストールやビルドは不要です。

- **そのまま開く**: `index.html` をブラウザで開く（`file://` でも動作）。
- **ローカルサーバ（推奨）**:
  ```bash
  cd health-tips-rating-app
  python3 -m http.server 8000
  # → http://localhost:8000/
  ```

## できること

- ジャンル別の **効くスコア順ランキング**（再生数＝バズに引きずられない透明なスコア）
- 詳細画面で **体感 / 科学的根拠 / 専門家監修を分離表示**（効能は断定しない）
- **やってみた → 体感フィードバック**（端末内に保存、スコアへ即反映）
- 貢献者への **チップ（デモ）**
- **日本語 / English** 切替
